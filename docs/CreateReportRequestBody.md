Can only be used with Intelligent Search Tracker metrics. * **&#x60;LISTINGS_LIVE_TYPE&#x60;**: The type of of listings live, either be &#x60;Claimed&#x60; or &#x60;Created&#x60; (can only be used with &#x60;LISTINGS_LIVE&#x60; metric). * **&#x60;LOCATION_IDS&#x60;**: Identify events by store ID of the corresponding location. * **&#x60;LOCATION_NAMES&#x60;**: Identify events by name of the corresponding location. * **&#x60;MEDIUM_ID&#x60;**: Identify the source of click this conversion is attributed to. Can only be used with Conversion Tracking metrics. * **&#x60;MONTHS&#x60;**: Identify events by months of the Gregorian calendar (January, February, etc.). * **&#x60;MONTHS_RETAIL&#x60;**: Identify events by retail month. Refers to the 4-5-4 merchandising calendar. * **&#x60;PARTNERS&#x60;**: The sites your reviews appear on. Can only be used with Reviews metrics. * **&#x60;PLATFORM&#x60;**: Groups data by the platform on which the action measured in **&#x60;metrics&#x60;** was conducted (e.g., Desktop, Mobile). * **&#x60;QUARTERS&#x60;**: Identify events by quarter. Only supported for Review metrics. * **&#x60;PRODUCT&#x60;**: Identify conversion analytics by the product in which they occurred. Can only be used with Conversion Tracking metrics. * **&#x60;PUBLISHERS&#x60;**: Groups data about the site-interaction events listed in **&#x60;metrics&#x60;** by the sites on which they occurred. * **&#x60;PUBLISHER_SUGGESTION_TYPE&#x60;**: The type of the publisher suggestion (can only be used with the &#x60;PUBLISHER_SUGGESTIONS&#x60; metric). * **&#x60;RATINGS&#x60;**: Can only be used with Reviews metrics except CUMULATIVE_RATING. * **&#x60;REVIEW_KEYWORDS&#x60;**: The keywords used in your reviews. * **&#x60;REVIEW_LABELS&#x60;**: Identify reviews by their label. * **&#x60;SEARCH_QUERY&#x60;**: Groups searches according to the search criteria used. Can only be used with the &#x60;SEARCHES&#x60; metric. * **&#x60;SEARCH_TERM&#x60;**: Raw Search Term entered by user for Search. * **&#x60;SENTIMENT_COLLECTION&#x60;**: Identify reviews by their sentiment collection of keywords. * **&#x60;STOREPAGES_DIRECTORY&#x60;**: The directories of your Store Pages.
